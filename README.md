# React Chess game

## Tech stack
 - React
 - Node.js
 - Redux

## Components
 - Menu
 - Game

## Features
 - Movement using click-to-select method
 - Player status
 - All piece logic including special moves like castling and en passant
 - Simple animation to move pieces
 - Cross-browser compatibility

## Prerequisites
 - Node - ^18.0.0

## Local setup
```
npm i
npm start
```

## Challenges faced
 - Implementing modular moves logic
